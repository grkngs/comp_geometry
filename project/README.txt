Initial(artificial) triangle's edges are not deleted. Ignore them.

For creating a point set with the size as the desired count, fill the box and click "Create Random Points and Run"

For inserting a point and creating the triangulation for the inserted points, click "Insert a Point" and "Run for manually inserted points".

You can not insert a point and also create a random point set.

Refresh the page to reset the current plane.
